# Ultimate Tic Tac Toe

Ultimate Tic Tac Toe is 9x9 tic tac toe game.

For this project we used the following technologies:
 - React Native (Android, IOS, Webapp)
 - Typescript
 - Excalidraw (/design)
 - Native Base
 - Eslint + prettier
 - ...
