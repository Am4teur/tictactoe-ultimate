# Documentation

## Initialization

Expo initialization of a React Native app with Typescript.

```
expo init -t expo-template-blank-typescript
```

## Project Structure

App.tsx
/src
 - /screens
 - /components
 - /utils
 - /assets

## Eslint and Prettier

```
npm add -D prettier
```

Add prettier [configuration](prettier.config.ts) file.
